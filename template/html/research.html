2. Ratings

Star ratings are symbols that indicates the quality or safety of listed things, such as accommodation, roads, or online listings. Star ratings can help businesses earn trust from potential customers, improve local search rankings, and boost conversions.
Requirements

Five Star rating system should be employed. They are usually displayed as a set of stars (usually 5) that can be clicked on to rate the product/service/content. The rating is then displayed as an average of all the ratings given by users.

Research

Daraz an e-commerce website was picked for the purpose of selection of rating system. In HTML and JavaScript, there are two ways to implement a star rating system: font style and image style. The font style uses a font that contains the star symbol and assigns different classes to each star based on the rating. The image style uses an image of a star and assigns different images to each star based on the rating.

Analysis
Source Website: Daraz
Lastrada
Similarities & Differences
Similarities Differences
Users, based on their experience can rate products they purchased. Daraz uses fontstyle from a third party source to style CSS. The star so used are images. Lastrada uses Emmet Abbreviation instead of fontstyles to style the CSS of the rating star
Conclusion
Both the image styling and font styling has its own perks and disadvantages. The font style is more flexible because it allows you to change the color and size of the stars easily while the image styling looks more appealing.

3. Add to Cart

In an ecommerce website, “Add to cart” is a feature that allows customers to add products to their online shopping cart when they click the “Add to cart” button. This feature acts as a mediator between the payment gateways and the ecommerce website. It aids the customers in acting upon their instinct buying behavior. It improves and enhances the site’s SEO and search rankings.
Requirements

A cart button is displayed at the bottom of image container. On clicking the cart button each product should be added to shopping list.

Research

An online shopping cart is software that allows customers to make purchases on a website. There are two types of shopping carts - hosted and self-hosted shopping carts. Hosted shopping carts are off-the-shelf solutions managed by the company who created them, while self-hosted shopping carts are open-source software that you can install on your own server Adorama an e-commerce website was picked for the purpose of selection of add to cart style.

Analysis

It would be better for growing company to outsource Hosted shopping carts from a third part as it includes Built-in functions and arrangement for the online transactions. While self-hosted shopping carts are benefitial for established companies which would allow to adhere the company to their established business rules and principles. open-source software that you can install on your own server

4. Form Validation

Form validation is a technical process where a web-form checks if the information provided by a user is correct. The form will either alert the user that they messed up and need to fix something to proceed, or the form will be validated and the user will be able to continue with their registration process Form validation is the process of checking whether the data entered by the user in a form is correct or not. It is required to prevent online form abuse by malicious users.
Requirements

A form that accepts different input from users should be accesible to users within the website. A message section should be present in the form where the users can write message or grief complain.

Research

There are two different types of client-side validation on the web: Built-in form validation and Javascript form validation. Built in form validation uses HTML form validation features by default. Amazon also an ecommerce website is from which reference of form validation is considered.

Analysis
Source Website
Lastrada
Similarities & Differences
Similarities Differences
Both websites records the input from users and check for form validation. In amazon a new display message appears in red to notify users of the missing data . Lastrada uses popup method to notify users about the missing data.
Conclusion
Improper validation of form data is one of the main causes of security vulnerabilities leading to attacks such as header injections, cross-site scripting, and SQL injections. Form Validation helps to prevent online form abuse by malicious users.
